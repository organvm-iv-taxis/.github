# organvm-iv-taxis/.github

Canonical organization metadata repo for
[ORGAN-IV: Taxis](https://github.com/organvm-iv-taxis).

This repository renders the public organization profile from
[`profile/README.md`](profile/README.md) and supplies org-wide GitHub community
health defaults for ORGAN-IV repositories that do not define their own local
files.
